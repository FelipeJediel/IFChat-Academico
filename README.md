# IFChat

Plataforma de fórum web desenvolvida para promover a comunicação, a colaboração e o compartilhamento de conhecimentos entre estudantes e membros da comunidade acadêmica do Instituto Federal de Educação, Ciência e Tecnologia do Pará (IFPA) – Campus Tucuruí.

## 📖 Sobre o Projeto

O IFChat é uma plataforma de discussão inspirada em fóruns e imageboards, adaptada ao contexto acadêmico. O sistema permite a criação de tópicos de discussão (*threads*), compartilhamento de informações e interação entre usuários por meio de comentários e categorias temáticas.

## ✨ Funcionalidades

- Cadastro e autenticação de usuários;
- Criação e gerenciamento de threads;
- Sistema de respostas e comentários;
- Categorias temáticas;
- Upload de imagens e anexos;
- Sistema de notificações;
- Interface responsiva.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- Git e GitHub
- Figma

## 🚀 Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/IFChat.git
```

2. Importe o banco de dados.

3. Configure as credenciais de acesso no arquivo de configuração.

4. Execute o projeto em um servidor local, como XAMPP ou Laragon.
